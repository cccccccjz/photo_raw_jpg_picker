# Photo RAW/JPG Picker

一个最初“随手 vibe coding”出来的小工具。

本来只是用来给自己挑照片，后来在小红书评论区发现也有人有同样需求，就把它整理完善了一下，做成了一个可直接使用的 GUI 小工具。

## 这个工具做什么

用于把照片目录里的 JPG 和 RAW 分开，并根据你筛选后保留下来的 JPG，自动同步对应同名 RAW。

典型场景：
- 你先用 JPG 快速挑片
- 再把挑中的 JPG 对应 RAW 一次性整理出来
- 减少手动找 RAW 的时间

## 支持的 RAW 格式

当前内置支持：

- .cr3
- .cr2
- .nef
- .arw
- .rw2
- .orf
- .raf
- .dng

对应代码变量是 `RAW_EXTS`（位于 `photo_sync_gui.py`）。

如果你使用其他型号相机拍出来的 RAW，只需要在 `RAW_EXTS` 里追加后缀即可，例如：

```python
RAW_EXTS = {".cr3", ".cr2", ".nef", ".arw", ".rw2", ".orf", ".raf", ".dng", ".你的格式"}
```

注意：后缀建议小写，并带 `.`。

## 使用方法

### 1. 启动程序

直接运行：

```bash
python photo_sync_gui.py
```

### 2. 在界面中按步骤操作

1. 选择照片根目录
2. 点击“1) 分开存储 JPG/RAW”
3. 到 `JPG_待筛选` 中手动删除不想要的 JPG（只保留想要的）
4. 回到程序点击“2) 根据筛选后的 JPG 同步 RAW”
5. 同名 RAW 会被移动到 `RAW_已同步`

程序会在根目录下使用/创建这些目录：

- `JPG_待筛选`
- `RAW_原片`
- `RAW_已同步`

## 打包 EXE

项目自带打包脚本：

```bash
build_exe.bat
```

双击运行后会自动：

1. 安装或升级 `PyInstaller`
2. 打包生成单文件 GUI 程序

打包结果在：

- `dist/PhotoPicker.exe`

## 环境要求

- Windows（GUI 使用 Tkinter）
- 已安装 Python（并可在命令行中使用 `python`）

## 小提示

- 这个工具按“同名文件（不含扩展名）”匹配 JPG 与 RAW。
- 当前默认处理的是所选根目录下的文件（不递归子目录）。
- 同名冲突时会自动重命名（如 `_1`, `_2`），避免覆盖已有文件。
